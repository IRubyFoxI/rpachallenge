## Мой проект - 

Это код, который сделан на PyCharm Community Edition 2023.3.5, написанный на Python. Там используется библиотеки, что буду указаны снизу

## Как работает?

Он берёт дравер microsoft Edge и запускает чистый браузер, после чего забирает с Excel данные, находит строчки по Path и вводит на сайте
